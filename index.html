Tristan Hertzog's Data Augmentor

Project was created for the purpose of being able to quickly customize augmentations, create unique data for your task, allow for tighter control for experiments and debugging, and to understand how augmentation works.

Libraries like Albumentations, torchvision.transforms, and Keras ImageDataGenerator support chaining augmentations easily and are recommended for many data augmentation tasks.

Instructions for Data Augmentor:
1. Select an image file (PNG, JPEG, SVG, GIF, WebP, or APNG)
2. Choose the number of images you would like to download
3. Check the options for image modifications
4. Select "Load Image"
5. Once ready to download, press the download image button

Modification Information:
When selecting one image, Horizontal and Vertical flips will always be applied, otherwise they happen 50% of the time.
