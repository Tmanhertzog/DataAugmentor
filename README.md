# Data Augmentor
### Created by: Tristan Hertzog

Demo video: [https://youtu.be/5kq_bNoCpoE](https://youtu.be/5kq_bNoCpoE)

A Web-based image augmentation tool to apply filters and generate batches of modified photos for download, a useful tool for data augmentation in image classification.

### Features:

* __Large Batch Downloads__: Allows for specified large batch downloads via a zip file with all applied mofications
* __Geometric Modifications__:  Image modifications including: Horizontal flip, vertical flip, rotation, cropping, translation, and shearing
* __Color Modifications__: Image modifications including: Brightness, contrast, saturation, invert colors, and blue tint
* __Noise Modifications__: Image modifications including: Gaussian noise and salt and pepper noise

### How to run:

**1.** Access the webpage [link here](https://tmanhertzog.github.io/DataAugmentor/)\
**2.** Select an image file (PNG, JPEG, SVG, GIF, WebP, or APNG).\
**3.** Choose the number of images you would like to download.\
**4.** Check the options for image modifications.\
**5.** press the download image button


### Blocked Download:
###### If your operating system has blocked the extraction of the zip file you have these options for possible solutions:

**1.** Under properties, select unblock in security\
**2.** In security settings mark file as an exception before extracting\
**3.** Compress zip into another format such as 7z and extract from there\
